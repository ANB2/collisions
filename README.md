Bike Accidents Map Visualization
================================

A time series visualization of cycling accidents in Montreal from 2006-2010.

Author: Corey Chivers

Contact: corey.chivers@mail.mcgill.ca

Date: 2012-09-09

	R CMD BATCH R/plot_accidents.R
